## ✨ Core Engineering & Architecture Features

* **🎛️ Advanced Error Correction Scaling:** Exposes fine-grained matrix configurations including `Q - reinforced` metrics, enabling codes to remain fully readable even if physically obscured or scratched.
* **📐 Custom Dimensional Scaling:** Dynamic dropdown matrix controller toggles internal resolution outputs (e.g., `180 px`) to output exact design requirements.
* **💾 Local Canvas Export Subsystem:** Integrated `DOWNLOAD PNG` utility captures the raw DOM canvas stream and forces a programmatic user download pipeline seamlessly.
* **📋 Clipboard Synchronization:** Instant `COPY CONTENT` utility hooks directly into the browser's Clipboard API to speed up continuous encoding workflows.
* **🎚️ Cyber-Matrix Dark UI Layout:** Obsidan black canvases paired with bright coral accents (`#f43f5e`), active state status nodes (`READY`), and responsive data cards.
